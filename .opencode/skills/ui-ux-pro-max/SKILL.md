## What I do

- Provide 67 UI Styles - Glassmorphism, Claymorphism, Minimalism, Brutalism, Neumorphism, Bento Grid, Dark Mode, AI-Native UI, and more
- Offer 161 Color Palettes - Industry-specific palettes aligned 1:1 with product types
- Curate 57 Font Pairings - Typography combinations with Google Fonts imports
- Support 25 Chart Types - Recommendations for dashboards and analytics
- Cover 15 Tech Stacks - React, Next.js, Astro, Vue, Svelte, SwiftUI, React Native, Flutter, HTML+Tailwind, shadcn/ui, etc.
- Provide 99 UX Guidelines - Best practices, anti-patterns, and accessibility rules
- Include 161 Reasoning Rules - Industry-specific design system generation

## How to use

### Design System Command (Advanced)

Use the skill to search for design intelligence:

```
/ui-ux-pro-max Build a landing page for my SaaS product
```

### Workflow Mode

Invoke the skill with your design request:
- Specify the type of interface you want to build
- Mention your target platform/framework
- Describe your target audience and use case

The skill will search its database and provide:
- Recommended UI style
- Color palette suggestions
- Font pairing recommendations
- Component patterns
- UX guidelines to follow
